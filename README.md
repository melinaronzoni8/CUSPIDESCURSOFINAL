# 🏔️ Cúspides UI - Módulo de Equipamiento Provisto

Este componente interactivo representa la sección "Material Incluido" del sitio web de Cúspides. Cuenta con una disposición vertical limpia donde el contenido textual se apoya sobre el marco estructurado de la aplicación y la gráfica se expande a pantalla completa.
